# 第三編　債権

# 民法 第三編「債権」 構造化要約

## 1. この部分の規律対象と位置づけ
本テキストは、日本民法の**第三編「債権」**である。これは、私法における財産関係の核心をなす規定であり、**債権者・債務者間の権利義務関係**を包括的に規律する。
*   **位置づけ**: 民法の第 2 編（物権）に次ぐ財産法の中核。契約、不法行為、不当利得、事務管理など、債権が発生・変動・消滅するすべての基本ルールを定める。
*   **規律対象**: 債権の目的、効力、多数当事者、譲渡、債務引受け、消滅、各種契約（売買、賃貸借、雇用など）、および契約以外の債権発生原因（事務管理、不当利得、不法行為）。

## 3. 重要な定義規定（条番号付き）
本編における主要な概念の定義は以下の通りである。

*   **債権の目的**: 第三百九十九条（金銭に見積もることができないものも目的となり得る）
*   **法定利率**: 第四百四条（年三パーセント）
*   **保証人**: 第四百四十六条（主たる債務者が債務を履行しないときにその履行をする責任を負う者）
*   **個人根保証契約**: 第四百六十五条の二（一定の範囲に属する不特定の債務を主たる債務とする保証契約で保証人が法人でないもの）
*   **定型約款**: 第五百四十八条の二（定型取引において、契約の内容とすることを目的として特定の者により準備された条項の総体）
*   **売買**: 第五百五十五条（財産権の移転と代金支払の約）
*   **消費貸借**: 第五百八十七条（種類・品質・数量の同じ物をもって返還をすることを約して物を受け取ること）
*   **賃貸借**: 第六百一条（物の使用収益と賃料支払の約）
*   **雇用**: 第六百二十三条（労働従事と報酬支払の約）
*   **請負**: 第六百三十二条（仕事の完成と報酬支払の約）
*   **委任**: 第六百四十三条（法律行為の委託と承諾）
*   **事務管理**: 第六百九十七条（義務なく他人のために事務の管理を始めた者）
*   **不当利得**: 第七百三条（法律上の原因なく他人の財産又は労務によって利益を受け、損失を及ぼした者）
*   **不法行為**: 第七百九条（故意又は過失によって他人の権利又は法律上保護される利益を侵害した者）

## 4. 中核的な規律内容（主要な条文番号）

### 第一章 総則
*   **債権の効力と債務不履行**: 履行遅滞（412）、履行不能（412 の 2）、損害賠償（415）、損害賠償の範囲（416）、金銭債務の遅延損害金（419）。
*   **債権者代位権**: 債権保全のため債務者の権利を行使できる（423）。
*   **詐害行為取消権**: 債権者を害する目的で債務者がした行為の取消し（424）。
*   **多数当事者**: 連帯債権・連帯債務（432-445）、保証債務（446-465）。特に個人根保証契約の極度額・元本確定期日制限（465 の 2-465 の 4）や事業保証の公正証書要件（465 の 6）が規定される。
*   **債権の譲渡**: 原則として譲渡可能（466）。対抗要件は通知または承諾（467）。
*   **債務の消滅**: 弁済（473）、相殺（505）、更改（513）、免除（519）、混同（520）。

### 第二章 契約
*   **契約の成立と効力**: 申込みと承諾（522）、同時履行の抗弁（533）、危険負担（536）。
*   **定型約款**: 不利益条項の無効（548 の 2）、変更の制限（548 の 4）。
*   **契約の解除**: 催告による解除（541）、催告によらない解除（542）、解除の効果（545）。
*   **各種契約**:
    *   **売買**: 担保責任（562-566）、危険負担（567）。
    *   **賃貸借**: 対抗力（605）、敷金返還（622 の 2）。
    *   **雇用**: 期間の定めのある雇用の解除（626）、やむを得ない事由による解除（628）。
    *   **委任**: 任意解除権（651）。

### 第三章 事務管理
*   管理者の義務（697）、緊急事務管理の責任制限（698）、費用償還請求（702）。

### 第四章 不当利得
*   返還義務（703）、悪意の受益者（704）、不法原因給付（708）。

### 第五章 不法行為
*   責任の発生（709）、責任能力（712-713）、使用者責任（715）、共同不法行為（719）、消滅時効（724）。

## 5. 罰則・制裁規定
*   **刑事罰**: 民法（第三編）には**刑事罰（罰金、懲役等）の規定は含まれていない**。
*   **民事制裁**: 違反した場合の制裁は民事的な効果として規定されている。
    *   **契約の無効・取消**: 定型約款の不当条項（548 の 2）、保証契約の方式違反（446、465 の 6）などにより効力を生じない。
    *   **損害賠償責任**: 債務不履行（415）、不法行為（709）、不当利得（703）などにより金銭賠償を命じられる。
    *   **強制執行**: 債務不履行の場合、裁判所による履行の強制が可能（414）。
    *   **時効**: 債権の消滅時効（724 など）により請求権が失われる。
